## Run Locally

Clone the project

```bash
  git clone https://github.com/sttvk/noface.git
  git clone https://github.com/sttvk/noface-api.git
```

Go to the project directory

```bash
  cd noface
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
